# DNS

Domain Name Server 域名伺服器

DNS主要用來做名稱與IP的轉換，讓使用者不必使用較難記憶的IP地址，改為使用較好記的名字使用網路上的各種服務，例如www.google.com的IP為142.251.42.228，一般人可能無法記住那麼長的數字，改為域名之後就變得相對好記
